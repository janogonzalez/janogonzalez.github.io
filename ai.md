# AI Usage

My personal projects are created for fun and/or learning. No generative AI is used for creating:
- This website
- My open source code
- My music, including cover art

There is one exception for this website: Occasionally, I may ask Google something like `"on my posts" vs "in my posts" correct grammar` which will show a generative AI summary. Given all the grammatical errors in my posts, you can see I rarerly do this.

Because of that, this website has a [`human.json`](/human.json) file according to the [human.json Protocol][protocol].

If I use generative AI tools for a new open source code project in the future, for example to understand better their usage, it will be explicitly mentioned.

## Work

I have used and will use generative AI tools at work for coding assistance, research and proofreading. That being said, I am against [Workslop][workslop].
